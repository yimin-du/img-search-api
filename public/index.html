FreeCodeCamp Project: Image Search API

User stories:

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.

Example usage:

https://custom-img-search.herokuapp.com/cute%20baby?offset=10

Expected Response: array of search results in JSON format.


https://custom-img-search.herokuapp.com/api/latest/imagesearch
